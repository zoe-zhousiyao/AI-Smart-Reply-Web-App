# 🤖 AI Smart Reply Web App

A simple full-stack web application that demonstrates how to integrate a Large Language Model (LLM) into a real-world system using the OpenAI API.

---

## 🚀 Features

- Input a message via a web interface
- Send the request to a Node.js backend
- Integrate OpenAI API for AI-generated responses
- Display dynamic replies in real time

---

## 🏗️ Tech Stack

- **Frontend:** HTML, JavaScript
- **Backend:** Node.js, Express
- **API:** OpenAI API (LLM integration)

---

## 🧠 How It Works

1. The user enters a message in the browser
2. The frontend sends a POST request to the backend (`/chat`)
3. The backend calls the OpenAI API
4. The AI generates a response
5. The response is returned and displayed in the UI
